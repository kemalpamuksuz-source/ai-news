### [Gemma 4: Byte for byte, the most capable open models](https://blog.google/innovation-and-ai/technology/developers-tools/gemma-4/)
**Source:** Google Blog | **Published:** April 2, 2026
**Summary:** Google DeepMind released Gemma 4 in four sizes (Effective 2B, Effective 4B, 26B MoE, 31B Dense) under an Apache 2.0 license — the first Gemma generation to ship fully open-source. The models support 256K context windows, native vision and audio, 140+ languages, and are optimised to run fully offline on edge devices such as phones and Raspberry Pi. Gemma 4 is available on Hugging Face, Kaggle, Ollama, Google AI Studio, and an AICore Android Developer Preview.
**Why it matters:** Gemma 4 is the most capable open-weights multimodal family to run on commodity hardware, raising the bar for on-device AI and directly challenging Chinese open-weights competitors.

### [Introducing the Agent Governance Toolkit: Open-source runtime security for AI agents](https://opensource.microsoft.com/blog/2026/04/02/introducing-the-agent-governance-toolkit-open-source-runtime-security-for-ai-agents/)
**Source:** Microsoft Open Source Blog | **Published:** April 2–3, 2026
**Summary:** Microsoft released the Agent Governance Toolkit (MIT license), a seven-package, multi-language (Python, Rust, TypeScript, Go, .NET) system that enforces policy at runtime before any agent action executes — with p99 latency under 0.1 ms. It is the first toolkit to address all 10 OWASP Agentic AI Top 10 risks and ships with integrations for OpenAI Agents SDK, LangGraph, LangChain, CrewAI, Haystack, PydanticAI, and Google ADK. Microsoft intends to transition governance to a community foundation.
**Why it matters:** As autonomous agents proliferate in production, a vendor-neutral, deterministic security layer covering the full OWASP agentic threat model fills a critical gap in the AI stack.

### [Introducing GPT-5.4](https://openai.com/index/introducing-gpt-5-4/)
**Source:** OpenAI | **Published:** March 5, 2026 (ongoing benchmark coverage)
**Summary:** OpenAI's GPT-5.4 is the first general-purpose AI model to surpass human performance on OSWorld-Verified (75.0% vs. the human baseline of 72.4%), a benchmark that measures autonomous desktop task completion via screenshots and keyboard/mouse actions. The model adds native computer use, a 1M-token context window, and a "Thinking" variant with integrated test-time compute. It is available via ChatGPT and API.
**Why it matters:** Crossing the human baseline on a real-world desktop benchmark is a concrete milestone for agentic AI, foreshadowing a near-term shift in how knowledge work is automated.
