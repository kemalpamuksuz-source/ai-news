### [Google TurboQuant Achieves 6x LLM Memory Compression at ICLR 2026](https://techcrunch.com/2026/03/25/google-turboquant-ai-memory-compression-silicon-valley-pied-piper/)
**Source:** TechCrunch | **Published:** March 25, 2026
**Summary:** Google Research's TurboQuant algorithm compresses large language model KV-cache to just 3 bits per value (down from 16), shrinking memory footprints by up to 6x and speeding up inference by up to 8x with no measurable accuracy loss. Using a two-step process — PolarQuant and the Quantized Johnson-Lindenstrauss transform — the method works as a post-processing step on existing models including Gemma and Mistral with no retraining required.
**Why it matters:** Dramatic inference-time memory reduction could substantially lower the cost and hardware requirements for deploying frontier models, rattling GPU memory chip valuations.
